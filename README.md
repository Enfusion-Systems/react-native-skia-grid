# react-native-skia-grid

Advanced Data Grid / Data Table for React Native via Skia

## Installation

```sh
npm install react-native-skia-grid
```

## Usage

```js
import { multiply } from 'react-native-skia-grid';

// ...

const result = await multiply(3, 7);
```
